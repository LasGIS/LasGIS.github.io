Объект “Layer”

Объект содержит описание строки тематического меню.

Свойства:

название доступ тип описание

IsVisible

 [rw] bool Признак видимости слоя (объекта) на карте.

IsMenu

 [rw] bool Признак заголовка слоя.

Name

 [rw] string Название строки тематического меню. В данной версии ограничено 24 символами

Alias

 [rw] string Альтернативное имя слоя. Складывается из типа объекта и номера типа объекта. Например: “П5” - пятая плоскость, “Т20” - двадцатая точка, “С19” - девятнадцатый слой.

Type

 [rw] short Тип объекта: 0 - плоскостной объект; 1 - линейный объект; 2 - точечный объект.

TypeNum

 [rw] short Номер типа объекта.

LayerNum

 [rw] short Номер тематического слоя.

BaseNum

 [rw] short Номер семантической базы.

ClassNum

 [rw] short Классификационный номер объекта.
