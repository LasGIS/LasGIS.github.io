Объект “Map”

Объект Map служит для визуализации карты на экране. Перед началом работы с объектом Map необходимо предварительно связать его с главным объектом Main. Объект Map является оконным объектом и может быть вставлен как “Control”.

Свойства:

Название [доступ] тип Описание
Level [rw] short уровень карты экрана.
MScale [rw] long масштаб вывода на экран.
AbsY1 [rw] double координата левой границы карты (запад) в метрах местности.
AbsX2 [rw] double координата верхней границы карты (север) в метрах местности.
AbsY2 [rw] double координата правой границы карты (восток) в метрах местности.
AbsX1 [rw] double координата нижней границы карты (юг) в метрах местности.
Interval [r] object MapObjectList список графических объектов, выводимых на карту.
Rastr [r] object RASTER_DRAWIN класс для рисования растра.
TwinkleMapOs [r] object MapObjectList список подсвеченных объектов.

Методы:

LinkMain (Main) связать объект Map с главным объектом Main.

Redraw () перерисовать карту.

GotoPoint (Level, MScale, AbsY, AbsX) перейти по координатам.

События:

MouseDown (AbsY, AbsX, X, Y, MouseKey) нажата кнопка мышки в окне карты.

MouseUp (AbsY, AbsX, X, Y, MouseKey) отжата кнопка мышки в окне карты.

MouseMove (AbsY, AbsX, X, Y, MouseKey) перемещение мышки в окне карты.

OnEndPaint () сообщение владельцу о завершении вывода карты.

Подробное описание свойств и методов:

Level

Текущий уровень карты “Level” может принимать значения от 1 до 9 включительно. При изменении значения текущего уровня карты карта будет автоматически перерисовываться.

Масштаб карты “MScale” может принимать значения от 20 до 100 000 000. или масштаба (MScale) карта будет перерисовываться.
