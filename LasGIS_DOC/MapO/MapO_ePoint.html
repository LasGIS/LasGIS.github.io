1.8.Формат элементарной точки (ePoint)

class ePoint {
   long   X;        // координата X точки в см. местности.
   long   Y;        // координата Y точки в см. местности.
   long   H;        // высота точки над уровнем моря (в см. местности).
   long   Prec;     // точность координат (в см. местности).
   short  Num;      // номер точки (общий для всего объекта).
   USHORT Branch:1; // признак примыкания к точке других линий, полос или плоскостей
   USHORT Blank:1;  // признак пропуска линии или границы от этой точки до следующей
   USHORT Free:14;  // пока пусто
}; // размер структуры 20 байт

Элементарная точка входит в состав элементарной линии графических объектов линия, полоса и плоскость.

Параметры <X> и <Y> составляют координаты точки объекта. Координата <X> определяет широту точки (снизу вверх на карте или с юга на север), а координата <Y> - долготу точки (слева направо или с запада на восток) в сантиметрах местности, если объект находится на уровнях от 0-го до 5-го (масштаб 1:200 000) включительно и в 1/10000000 долях градуса, если объект находится выше 5-го уровня.

<Prec> - точность координат <X> и <Y> определяется как погрешность определения координаты в см. местности. Точность определяется:

при скалывании мышкой - автоматически по масштабу изображения на экране,

при вводе с клавиатуры – погрешность считается нулевой.

<Num> - определяет номер данной точки, применительно для всего объекта. Номер точки может быть полезен при сборе объекта из частей с разных планшетов, а также участвует в документах при определении границ землепользования.

Флаг <Branch> указывает на то, что в данной точке к данному объекту примыкает другой объект с теми - же свойствами например смежный земельный участок или ответвление к трубопроводу). Этот признак поможет при решении сетевых задач.

Флаг <Blank> указывает на то, что следующая линия объекта не показывается на карте. Это необходимо, например, при рисовании отдельных участков реки или дороги.
