Что такое HTML

Изначально HTML задумывался как формат документов с сохранением разметки. Потом, к нему добавили динамичность через возможность программно изменять атрибуты. Однако если забыть про эту возможнось - HTML так и остался документом с разметкой.

1. HTML состоит из тегов

Тег это такая штука, которая начинается конструкции <название> и кончается конструкцией c тем-же названием, но с косынкой впереди названия тега </название>.

Есть укороченный тип тега. В этом случае косынка располагается в конце первой конструкции, а замыкающей конструкции вовсе нет - <название_тега название_атрибута="описание атрибута"/>. Укороченный тип обычно применяется с атрибутами.

Тег может содержать некоторые аттрибуты. В этом случае в первую конструкцию добавляют описание одного или нескольких атрибутов - название_атрибута="описание атрибута":

<тег1 attrib1="значение первого атрибута" attrib2="значение второго атрибута" > Здесь располагается тело тега, которое может кроме текста содержать и внутренние теги. Например: <тег2 attrib="значение атрибута" ></тег2> <тег2 attrib="значение атрибута" /> Последний тег является укороченным. </тег1>

1.1. Пробелы и специальные символы

Лишние пробелы и переводы строк игнорируются. Это означает, что сколько бы ни поставить пробелов в исходном документе, "броузер" поставит только один пробел. Преводы строк будут располагаться в зависимости от размеров окна и правил форматирования абзаца.

Для принудительной вставки пробела применяют специальный символ &nbsp; Все специальные символы начинаются с амперсанта - & и кончаются точкой с запятой - ;

Вот далеко не полный список специальных символов:

в исходнике в броузере описание
&nbsp; [ ] неразделимый пробел
&lt; < знак меньше
&gt; > знак больше
&amp; & амперсант

2. Минимальный набор для создания странички

Описание полного набора тегов можно найти в HTML 4.01 Specification (W3C).

Здесь будут только теги, необходимые для создания документа.
2.1. <P> Абзац

Абзац является основным способом формирования HTML документа. Может применяться с разными стилями. Категорически не рекомендуется использовать прямое (помимо стиля) форматирование абзаца.

<P class="Prog">Основной текст.</P>

2.1.1 Допустимое форматирование шрифта.

Иногда нужно выделить или подчеркнуть отдельные слова или словосочетания. В этом случае применяется теги форматирования шрифта:

в исходнике в броузере
<b>толстый</b> шрифт толстый шрифт
шрифт c <i>наклоном</i> шрифт c наклоном
шрифт c <u>подчёркиванием</u> шрифт c подчёркиванием
шрифт c <b><i><u>тем и другим одновременно</u></i></b> шрифт c тем и другим одновременно
2.2. <H1> - <H6> Заголовок

Заголовки разных уровней. Использование стилей возможно, но в данном случае не имеет смысла!

<H1> - имеет наивысший уровень, а <H6> - соответственно наинизший.

в исходнике в броузере

2.3. <A> Ссылка

Ссылка используется для перехода на другой документ HTML. Для задания условий перехода используются аттрибуты тега:

имя атрибута   пример значения описание
href = Explanation.html#mark Ссылка на тот документ, на который будет переход после клика. Можно после знака # указать метку, к которой перейдёт браузер. Кроме того, в качестве ссылки может быть вызов функции JavaScript.
target = target_doc Объект типа frame в котором будет показываться выбранный документ.
name = mark Используется для обозначения метки, к которой перейдёт браузер

Всё что находится внутри тела тега (от > до </a>) будет являться ссылочной строкой. Таким образом ссылкой может быть текст, картинка, текст + картинка и т.д.

Для указания на конкретное место в страничке необходимо сделать метку в данной страничке, после чего можно на эту метку указывать.

2.3.1. Создание метки

Для создания метки надо заполнить только атрибут name:

<a name="mark">Метка внутри документа Explanation.html</a>

Метка внутри документа Explanation.html
2.3.2. Создание ссылки внутри документа

Для создания ссылки на метку внутри одного документа в атрибуте href пишем только метку:

<a href="#mark">Ссылка на метку внутри документа</a>

Ссылка на метку внутри документа
2.3.3. Создание ссылки из другого документа

Для создания ссылки на метку из другого документа необходимо указать сам документ:

<a href="Explanation.html#mark">Ссылка на документ Explanation.html + метка</a>

Ссылка на документ Explanation.html + метка
2.3.4. Добавляем цель (target)

Добавляем цель для документа (объект типа frame, в котором должна отобразиться страничка):

<a href="Explanation.html#mark" target="target_menu">Ссылка на документ с выводом в target_menu</a>

Ссылка на документ с выводом в target_menu
2.3.5. Создаём ссылку через вызов функции

Для вызова через функцию надо предварительно функцию создать. В нашем случае функция находится в файле web/images/menu.js и называется LoadUrl().

Первый параметр содержит ссылку на документ, который затем поместит в окно документа (target="target_doc").

Второй параметр содержит ссылку на меню. Меню это то-же HTML документ. Функция LoadUrl() поместит это меню в левое маленькое окно, где по определению должно быть меню.

Третий параметр содержит строку заголовка. В момент выполнения функции эта строка помещается в заголовок всей страницы.

Главное помнить, что эта функция должна работать только в меню и главном окне (index.html). В документ её нельзя помещать:

<a href='javascript:LoadUrl( "Explanation.html#mark", "ExplanationMenu.html", "Перешли по ссылке через вызов функции" )'>Ссылка через вызов функции</a>

Ссылка через вызов функции
2.4. <img> картинка

Предварительно картинку надо нарисовать в одном из удобоваримых форматов. Здесь её можно только показать. Тег <img> является укороченным, поэтому он содержит только первую конструкцию (без </img>)

Картинку можно ужать или растянуть. Для этого используют атрибуты высота - height и ширина - width. Размерные атрибуты могут указываться либо в пикселах ('10px' - 10 пикселов), либо в процентах от размера экрана ('10%' - 10 процентов) при этом, почему-то, имеется в виду только ширина клинтской части отображаемого окна. Как для ширины, так и для высоты картинки.

2.4.1. Просто картинка

Выводим простую картинку:

<img src="images/TradeMark.jpg" height='64px' width='64px' />

2.4.2. Картинка как ссылка
Кроме этого картинка может быть ссылкой. Для этого её надо обрамить тегом <a>. Однако, никто не мешает добавить текст после картинки.

<a href="#mark"><img src="images/TradeMark.jpg" /></a>

2.4.3. Убираем бордюр
Если убрать бордюр, то какртинка - ссылка будет без рамки:

<a href="#mark"><img border='0' src="images/TradeMark.jpg" /></a>

2.5. <table> Таблица

Таблица состоит из строк (тег <tr>) и значений столбца для данной строки (тег <td>). Каждая таблица содержит набор строк, а каждая строка - набор значений столбцов. Кроме того, существует тег <th>, который применяетс как тег <td>, но означает заголовок. Впрочем, тег <th> может располагаться в любом месте таблицы.

<table border='1'>
  <tr><th>заголовок 1</th><th>заголовок 2</th><th>заголовок 3</th></tr>
  <tr><td>данные 11</td><td>данные 12</td><td>данные 13</td></tr>
  <tr><td>данные 21</td><th>данные 22</th><td>данные 23</td></tr>
  <tr><td>данные 31</td><td>данные 32</td><td>данные 33</td></tr>
</table>
заголовок 1заголовок 2заголовок 3
данные 11данные 12данные 13
данные 21данные 22данные 23
данные 31данные 32данные 33
2.5.1. Совмещение столбцов и строк

Для совмещения двух или более соседних столбцов в одной строке, вместо этих столбцов рисуется один столбец (тег <td>), в котором указан атрибут colspan='2'.

Для совмещения двух или более соседних колонок в одну колонку, необходимо в самом верхнем значений столбца (теги <td>) указать атрибут rowspan='3', а остальные значений столбца удалить. См. пример:

<table border='1'>
  <tr><th>заголовок 1</th><th colspan='2'>заголовок 2, 3</th></tr>
  <tr><td>данные 11</td><td>данные 12</td><td rowspan='3'>данные 13, 23, 33</td></tr>
  <tr><td rowspan='2'>данные 21, 31</td><th>данные 22</th></tr>
  <tr><td>данные 32</td></tr>
</table>
заголовок 1заголовок 2, 3
данные 11данные 12данные 13, 23, 33
данные 21, 31данные 22
данные 32
2.5.2. Совмещение столбцов и строк

Для совмещения двух или более соседних столбцов в одной строке, вместо этих столбцов рисуется один столбец (тег <td>), в котором указан атрибут colspan='2'.
