Программный комплекс "LasGIS"

Программный комплекс LasGIS существует с 1992 года как информационно-поисковая система городского хозяйства, как основа муниципальной ГИС.

В данный момент с комплексом работают различные геодезические (и не только) организации города Омска:

  • ТИСИЗ
  • 152 экспедиция
  • Главомскархитектура
  • Аграрный Университет и т.д.
  • Департамент Недвижимости
  • Городской Земельный комитет

Комплекс предназнач для работы с глобальным банком кадастровой информации в режиме распределенного доступа. Банк кадастровой информации состоит из трех частей:

  • Растровая карта
  • Векторная карта
  • Семантическая база

Состав ПК LasGIS

В состав программного комплекса LasGIS входят следующие части:

Geo.exe (LasGIS под DOS)

Основная программа для работы в среде MS DOS имеет следующие возможности:

  1. редактирование графических объектов:
    1. мышкой (по растровой карте),
    2. с помощью дигитайзера,
    3. через занесение координат точек вручную;
  2. Создание и отладка программ на встроенном языке описания задач;
  3. работа с атрибутивной базой данных (получение справки по объекту, поиск графического объекта по атрибутам);
  4. возможность конвертировать графические и семантические данные в другие системы;
  5. настройка тематических слоев карты;
  6. настройка вида изображения;
  7. настройка семантических баз данных;
  8. настройка номенклатуры планшетов
  9. имеется защита от несанкционированного доступа (система паролей и журнал регистрации действий пользователей)
  10. печать карт на известных принтерах;
  11. имеется интерактивная подсказка по всем функциям и режимам системы.

Для ознакомления с программой Geo.exe можно воспользоваться демонстрационным роликом.

Скачать деморолик DEMOROL.zip (1.2 Mb)

После скачивания, архив надо развернуть в отдельном каталоге и запустить файл play.bat.

Более подробно про работу с роликом можно посмотреть в файле DEMOROL.TXT, находящемся внутри архива.

LasGIS.exe (LasGIS под Windows)

Основная программа для работы с картой в среде Windows 95/NT/2000/XP. Имеет следующие возможности:

  1. печать высококачественных карт на любых принтерах любого формата с неограниченным числом страниц;
  2. редактирование графических объектов;
  3. получение справки по объектам (из атрибутивной базы).
  4. Составление запросов с поиском по атрибутивной базе данных.
PCXBreak.exe (MS DOS)

Предназначена для создания бесконечнах растровых карт системы LasGIS.

Преобразует входные файлы .PCX в файлы растровой карты с возможностью ограничения трансформируемой области.

MakeRastr.exe (Windows)

Как и PCXBreak.exe предназначена для создания бесконечнах растровых карт, но имеет следующие отличия:

  1. добавлена поддержка форматов .PCX и .BMP.
  2. работает значительно быстрее за счет эффективного использования памяти и работы в 32-разрядном режиме.
  3. добавлена функция очистки исходного растрового изображения.
